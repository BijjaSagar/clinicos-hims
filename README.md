# clinicos-hims

Production **Laravel application** code lives in **`backend/`**.

- Install: `cd backend && composer install`
- Env: copy `backend/.env.example` → `backend/.env`
- Deploy web root to `backend/public/`

Docs, datasets, mobile apps, and other folders may exist on your machine beside this clone; **they are intentionally not part of this Git repository** (backend-only).
